## How to Run the Game

1. **Clone the repository:**
    ```bash
    git clone https://github.com/Hamodea/Breakout_Game.git
    cd breakout-game
    ```

2. **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the game:**
    ```bash
    python breakout_game.py
    ```

Enjoy the game!
